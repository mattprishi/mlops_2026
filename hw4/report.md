# Отчётик

Ошибки и устойчивость

Невалидный JSON или типы по схеме — 422. Отрицательный возраст и подобное — 422. Не хватает фич для текущей модели или неизвестные имена фич — 400. На /updateModel: пустой run_id — 422; нет run или модели в MLflow — 404 или 503, старая модель остаётся; проблемы с чтением артефакта — 503, модель не меняется. Если модель не загружена — 503 на predict. Если вероятность не число — 500.

Тесты

В tests на pytest: предобработка to_dataframe (колонки, пропуски, лишние имена), отрицательный возраст, predict на мок-модели (вероятность 0–1, класс), health и predict с хорошим и плохим телом, updateModel с моком load_model (смена модели и откат при ошибке MLflow), сценарий predict — updateModel — predict.

Метрики

GET /metrics, порт 8890. Считаю запросы и латентность HTTP, CPU и память процесса, время предобработки и инференса, гистограммы по числовым фичам и по вероятности, счётчики предсказаний по классу, счётчики обновления модели, Info с run_id, типом модели и списком фич. Перцентили по времени в Grafana через histogram_quantile по bucket.
